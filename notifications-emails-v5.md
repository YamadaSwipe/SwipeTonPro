## 📄 **FORMULAIRE DE CONSENTEMENT DE VERSEMENT**

### **Intégration dans le Dashboard Particulier**
```javascript
// Dans dashboardService.ts
const generateConsentForm = (projetId, proId, montant) => {
  return {
    titre: "Formulaire de consentement de versement",
    projet: projetId,
    professionnel: proId,
    montantTotal: montant,
    dateGeneration: new Date(),
    urlDownload: `/api/documents/consent-form/${projetId}`,
    statut: "disponible"
  };
};
```

### **Structure du formulaire**
```
📄 FORMULAIRE DE CONSENTEMENT DE VERSEMENT

Projet : [Titre projet]
Date : [Date]
Particulier : [Nom particulier]
Professionnel : [Nom professionnel]

🔒 CONSENTEMENT DE SÉCURISATION DES FONDS

Je soussigné(e) [Nom particulier] consens à :

□ Sécuriser l'acompte de €[montant] sur Stripe
□ Sécuriser le montant total de €[montant] sur Stripe
□ Verser par paliers : [détail des paliers]

📋 CONDITIONS :
- Les fonds seront débloqués selon les étapes validées
- Protection mutuelle des deux parties
- Médiation SwipeTonPro en cas de litige

Signature : _________________________
Date : _________________________
```
